# Version 0.8.0.0 - Chutes and Ladders
- release for KSP 1.11.x
- add engine to Restock whitelist (default off)
- add Restock patch
### Added Firespitter.cfg patch
 - moved Firespitter wheel code from wings and gear to patch
 - replaced with ModuleAnimateGeneric
 - this is only to address the bouts of altitude sickness and diziness from vessels spawning 1000 meters above the covers.
### [pmv-engine] miniPuff 
  - make it multi-mode
  - add attachment node to top
  - adjust attachment and bulkheadProfiles
-  wings and body - add attachment nodes
### add parts:
  - mini-RadialDrogue-rect
  - mini-RadialDrogue-rnd
  - mini-RadialChute-rect
  - mini-RadialChute-rnd
### Adjust pmv-shuttle-body CoMOffset (from 0.30 to 0.32) and CoLOffset (from 0.030 to 0.33)
